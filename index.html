Introduction

Java is a strongly-typed cross-platform compiled object-oriented programming language. It is a large and heavyweight language. Java was built on the principle of write once, run everywhere.

Java code is compiled to what is known as bytecode and this bytecode is what is intepreted by the Java virtual machine (JVM). This bytecode is what give Java its security and portability. Java can be used to write about any software from desktop app, mobile apps, webservers, software for automation systems etc.
Some words that have been used to describe java are secure, Distributed, architecture neutral, object-oriented, Multithreaded, interpreted, etc

Data types

Java has eight primitive data types namely: byte, short, int, long, char, float, double and boolean.

Methods

Methods in some other languages are called subroutines or functions.

  • Syntax of methods

    modifier returntype methodname (argument list){ statements; ... [return variable]; }

    Modifiers determine the scope or accessibility of a class or variable, how it can be accessed by the program.

      modifiers are
    • public
    • private
    • static
    • protected
    • volatile
    • friend

    Return type determines what type of value(data type) will return to where the method was called from.

      Return types are
    • int
    • String
    • double
    • float
    • long
    • boolean
    • char
    • void

    Method name is any name you decide to identify your method by.

    Arguments are the things you pass into the method that could be made us of inside the method

    Example method for calculating simple interest

    public class MethodSimpleInterest{ public static void main(String[]args){ int principal = 10000; int rate = 2.5; int time = 5; double receivedInterest = modifiedInterest( principal, rate, time); System.out.println("Interest = " + receivedInterest); } public double modifiedInterest(int p, int r, int t){ double intval = (p*r*t)/100; return intval; } }
